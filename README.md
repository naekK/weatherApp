# weatherApp
現在地の天気をブラウザに表示するアプリ

ブラウザを開いたときに現在地の緯度・経度を取得し、

取得した緯度・経度からOpenWeatherOrgのAPIを使って現在地の天気情報をJSONで取得

その結果を表示しています
